==================================
NASA IXS ENTERPRISE WARP SHIP -----BETA-----
==================================
Original design and concept - Mark Rademaker
Creating for Kerbal Space Program by Stevie_D

This pack contains only spacecraft parts to create the Kerbal version of the IXS class for endgame KSP-Interstellar.
This BETA version includes-
- Command Module
- Main Hull/Antimatter Generator and Container.
- New Alcubierre Warp Ring and Warp Ring Electrical Generator
- Quantum Vacuum Thrust Engine


To come in Beta 2
- Crew Module
- Science Module
- 2 IXS Communotrons
- 2 Fuel Tanks (LFO and Karbonite)
- 5 IXS versions of science equipment (Gravioli Detector, Thermometer, Magnotometer, Gamma Ray Spectrometer, Science Junior, Karbonite Detector, Communotron)

Dependancies :
KSP-Interstellar
Firespitter
(see Kerbal forums for both mods)


============
INSTALLATION
============

To install, place the GameData folder inside your Kerbal Space Program folder. If asked to overwrite files, do so. 

As this pack bundles Firespitter, please make sure to check you always have the latest up to date version.
